## Min Syntax Notes

Attributes
```
@Special
@SpecialAttribute
@Special("val")
@assembly:Special
```

Loops
```
for i in 10
    ; 0-9

for i in myList
    ; iterate over myList

for i in myList
    Console.Write(i)
bet
    Console.Write(", ")

for i = 0, i < 7, i += 1
    ; normal for loop

for
    ; infinite loop

for i < 7
    ; while loop
```
